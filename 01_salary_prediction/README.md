# Salary Prediction using Linear Regression

## 📌 Objective
The goal of this project is to predict salary based on years of experience using a machine learning model.

---

## 📊 Dataset
- Feature: Years of Experience  
- Target: Salary  
- Simple dataset used for understanding regression fundamentals  

---

## ⚙️ Workflow
1. Data Loading using Pandas  
2. Data Analysis (EDA)  
3. Train-Test Split  
4. Model Training using Linear Regression  
5. Model Prediction  
6. Evaluation using MAE and R² Score  
7. Data Visualization using Matplotlib  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib
